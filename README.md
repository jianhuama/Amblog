#### 安装教程

1. git clone https://gitee.com/Jack-chendeng/Amoblog-front.git
2. cd Amoblog-front
3. npm install
5. 修改您本地ServerUtil.js 中服务器请求地址：baseURL
4. npm run dev

#### 使用说明

1. 建议使用 pm2 进行运行 使用命令
2. 线上进入目录 执行 pm2 start npm --name "main" -- run build
3. pm2 logs 可查看启动日志
